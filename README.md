# Personal Expense Tracker

Веб-приложение для учета личных финансов, созданное на базе Django. Проект позволяет отслеживать расходы, управлять бюджетами и осуществлять переводы между счетами с автоматическим пересчетом баланса.

## Ключевы фичи
* **Управление счетами:** Создание нескольких счетов с отслеживанием актуального баланса.
* **Транзакции и переводы:** Фиксация расходов по категориям и переводы между собственными счетами.
* **Транзакционная целостность:** Вся математика изменения балансов реализована через сигналы с использованием `transaction.atomic()` на уровне базы данных.
* **Контроль бюджета:** Установка лимитов на категории трат. Система автоматически предупреждает пользователя, если расходы превышают 80% от заданного лимита.
* **Умная история операций:** Продвинутая фильтрация транзакций по датам, счетам и категориям с использованием `django-filter` и постраничной навигацией.
* **Изоляция данных:** Строгое разделение данных между пользователями 

## Технологический стек
* **Backend:** Python 3, Django 6
* **Database:** PostgreSQL
* **Frontend:** HTML/CSS, Bootstrap 5, crispy-forms
* **Tools:** django-filter, python-dotenv (для управления секретами)
